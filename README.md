# ⏱️ Stopwatch App - SCT_WD_2

Interactive stopwatch built as Task 2 for **SkillCraft Technology** Web Development Internship.

### 🔗 Live Demo
https://kommuriyamini451.github.io/SCT_WD_2/

## ✨ Features
- **Start / Pause / Reset** functionality
- **Lap Timer** - record multiple lap times
- **Millisecond Precision** - accurate to 0.01 seconds  
- **Keyboard Shortcuts**
    - `Space` - Start / Pause
    - `L` - Record Lap
    - `R` - Reset
- **Modern UI** - Glassmorphism design with sky blue gradient
- **Circular Buttons** - Dark aesthetic theme
- **Responsive** - Works on mobile, tablet & desktop

## 🛠️ Tech Stack
- **HTML5** - Structure
- **CSS3** - Glassmorphism, Flexbox, Grid, Animations
- **JavaScript** - setInterval, DOM manipulation, Event handling
